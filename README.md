# 📘 DSA in Java
 
A personal repository for learning and practising **Data Structures and Algorithms** implemented in Java. This is not a library or a framework — it's a structured notebook of my understanding, written in code.
 
Every file here is written from scratch to reinforce concepts, not just to produce working code.
 
---
 
## 🎯 Why This Repo Exists
 
Understanding DSA deeply is one of the most high-leverage skills in software development — it sharpens problem-solving, improves code quality, and is central to technical interviews. This repo is my way of learning by doing: writing each data structure and algorithm by hand, understanding *why* it works, and keeping notes on trade-offs and complexity.
 
If you've stumbled upon this, feel free to use it as a reference — but the primary audience is future me.
